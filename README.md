# Ideas Notes

## Идея проекта

Веб-приложение "Ideas Notes" — это платформа для обмена заметками, где пользователи могут публиковать свои текстовые заметки, комментировать заметки других и отмечать понравившиеся записи.

## Ключевые сущности

- **Пользователи (Users)**: Учетные записи пользователей с данными для авторизации (логин, пароль, email) и персональными настройками.

- **Заметки (Notes)**: Текстовые записи, содержащие заголовок, текст, автора, дату создания и (опционально) категорию(возможно также фотографии к заметкам).

- **Комментарии (Comments)**: Текстовые сообщения, привязанные к конкретной заметке, с указанием автора и даты.

- **Лайки (Likes)**: Отметки, которые пользователи ставят понравившимся заметкам.

- **Избранное (Favorites)**: Список заметок, сохраненных пользователем для быстрого доступа.

## Основные рабочие страницы

- **Главная страница (Home Page)**: Отображает ленту всех общедоступных заметок с возможностью фильтрации по категориям или популярности (по количеству лайков).

- **Страница заметки (Note Page)**: Детальная страница конкретной заметки с текстом, комментариями и кнопками для лайков/добавления в избранное.

- **Профиль пользователя (Profile Page)**: Страница с информацией о пользователе, его заметками и статистикой активности (например, количество лайков).

- **Страница авторизации (Auth Page)**: Формы для регистрации и входа в систему.

## Функциональность

- **Авторизация и регистрация**: Пользователи могут создавать учетные записи, входить в систему и управлять своим профилем.

- **Управление заметками**: Создание, редактирование и удаление заметок с поддержкой категорий и тегов для удобной организации.

- **Взаимодействие пользователей**: Возможность комментировать заметки других пользователей, ставить лайки и добавлять заметки в избранное.

- **Фильтрация и поиск**: Поиск заметок по ключевым словам и фильтрация по категориям или популярности.
